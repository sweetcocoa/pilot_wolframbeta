## Wolframbeta ##

### 구현된 기능 ###
- 사칙연산
    두 실수의 사칙연산
    괄호를 포함한 여러 실수의 사칙연산
    괄호, 곱셈, 덧셈을 포함한 여러 실수의 연산자 우선순위를 따르는 사칙연산
- 거듭제곱
    거듭제곱의 우선순위는 괄호 > 거듭제곱 > 곱셈, 나눗셈
- 연산 트리 생성
- 일반함수의 토큰화
- pi, e 등의 수학 상수 지원(부동소수점 실수 근사값)
- 초등함수 (대수함수, 삼각함수, 지수함수, 로그함수)
- 합성함수
